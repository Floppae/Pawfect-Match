🚀 Getting Started (Local Development)
1. Clone the repo
git clone https://github.com/username/Pawfect-Match.git
cd Pawfect-Match/client
2. Install dependencies
npm install
3. Run the app locally
npm start
The app will open at http://localhost:3000.
